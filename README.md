# Description

This is a sample Jiangren CMS.

# Pre-requiste
1. Install docker-compose
2. Install docker

# Run locally
```
docker-compose up
```
Access via http://127.0.0.1

# Development
```
docker-compose up
cd app
pip install -r requirements.txt
```
Open the project in your IDE (e.g. vscode) and run app.py.

Access via http://127.0.0.1:8080
